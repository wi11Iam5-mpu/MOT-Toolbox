# MOT-Toolbox
This is a simple toolbox for MOT.

# 
``
project/
│
├── main.py              # 主程序入口
├── video_player.py      # 视频播放器模块
├── modules/             # 功能模块目录
│   ├── __init__.py      # 模块目录初始化文件
│   ├── base_module.py   # 基础模块类
│   ├── video_overlay.py # 视频分屏模块
│   ├── video_annotation.py # 视频标注模块
│
├── config.json          # 配置文件
└── annotations/         # 示例标注文件目录
    └── example.txt
``
